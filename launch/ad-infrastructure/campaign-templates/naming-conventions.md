# Campaign Naming Conventions

## Format

```text
{PLATFORM}_{OBJECTIVE}_{COUNTRY}_{PRODUCT}_{AUDIENCE}_{VERSION}
```

## Examples

- `TTK_CONV_US_FURLIFT_COLD_V1`
- `TTK_CONV_US_FURLIFT_RETARGET_V1`
- `META_SALES_US_FURLIFT_IGREELS_V1`
- `META_SALES_US_FURLIFT_RETARGET_V1`

## Ad Set Format

```text
{PLATFORM}_{COUNTRY}_{PLACEMENT_OR_AUDIENCE}_{EVENT}_{VERSION}
```

Examples:

- `TTK_US_PETOWNERS_ATC_V1`
- `META_US_IGREELS_ATC_V1`
- `META_US_SITEVISITORS30D_PURCHASE_V1`

## Ad Format

```text
{PLATFORM}_AD_{ANGLE}_{CREATIVE_VERSION}
```

Examples:

- `TTK_AD_PROBLEM_FIRST_V1`
- `TTK_AD_POV_STYLE_V1`
- `META_AD_TRANSFORMATION_FIRST_V1`
