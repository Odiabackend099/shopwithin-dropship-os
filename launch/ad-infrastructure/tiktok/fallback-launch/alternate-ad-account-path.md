# Alternate TikTok Ad Account Path

Use this only if the operator wants a second legitimate attempt at TikTok Ads Manager while the company account is blocked.

## Path

1. Create a fresh TikTok Ads Manager account directly from TikTok, not from the Shopify app.
2. If TikTok offers account type selection, choose the truthful operator type:
   - Individual / sole proprietor if using personal identity.
   - Business only if using `ODIADEV AI LTD`.
3. Do not use fake country, address, or business identity.
4. If TikTok requests individual Tax ID, retrieve it from the official Nigerian Tax ID Portal using the operator's NIN. The operator must personally enter NIN and approve consent.
5. If the new ad account reaches `Good to go` status, create a new TikTok pixel.
6. Install the new pixel in Shopify Customer Events as a custom pixel.
7. Verify:
   - Pageview
   - ViewContent
   - AddToCart
   - InitiateCheckout
   - Purchase through Flutterwave sandbox
8. Launch a tiny traffic/conversion test only after the pixel fires.

## If TikTok Blocks Again

Stop. Do not keep creating accounts repeatedly. Repeated failed advertiser accounts can increase platform risk.

## First Campaign If It Works

- Campaign objective: Website conversions if pixel purchase/checkout events verify; otherwise traffic.
- Daily cap: NGN 3,000-5,000.
- Country: start with lowest CPC English-speaking tests only if TikTok allows targeting.
- Creative count: 3 videos.
- Kill after NGN 2,000-3,000 if there is no site traffic.
