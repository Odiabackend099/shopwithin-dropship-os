# Metrics Checklist

## Traffic Metrics

- Impressions.
- Reach.
- CPM.
- Link CTR.
- CPC.
- Landing page views.

## Store Metrics

- Sessions by UTM source.
- Product page views.
- AddToCart count.
- AddToCart rate.
- Checkout starts.
- Purchase count.
- Conversion rate.

## Money Metrics

- Spend.
- Revenue.
- Cost per AddToCart.
- Cost per checkout start.
- CAC.
- ROAS.
- Estimated gross margin.

## First Target Benchmarks

These are early directional targets, not guarantees.

| Metric | First Target |
| --- | --- |
| Link CTR | `>= 1%` |
| CPC | `< USD 1` equivalent |
| AddToCart rate | `>= 3%` |
| Checkout-start rate | `>= 1%` |
| First purchase | After enough traffic, not expected from the first few clicks |
