# Source Notes

These notes summarize the official platform guidance used for this launch package.

## Shopify / Meta

Shopify's Facebook and Instagram by Meta sales channel can connect products/catalogs and Meta Marketing assets. Shopify data-sharing settings can connect Meta Pixel and Conversions API. Shopify lists the relevant events as:

- PageView
- ViewContent
- Search
- AddToCart
- InitiateCheckout
- AddPaymentInfo
- Purchase

Shopify notes that Conversions API sends Purchase between Shopify and Facebook servers and that data-sharing level is an operator privacy choice.

## Instagram

Instagram Shopping redirects customers to the Shopify online store checkout.

## TikTok

TikTok Promote is a separate low-friction path from full TikTok Ads Manager, but it must be tested in the operator's TikTok mobile app. Stop if it triggers the same business/tax verification blocker.
