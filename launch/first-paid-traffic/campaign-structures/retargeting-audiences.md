# Retargeting Audiences

Create these after Meta Pixel is connected.

## Website Visitors

Audience:

```text
META_RT_WebsiteVisitors_30D
```

Rules:

- All website visitors.
- 30-day window.
- Exclude purchasers if Purchase event exists.

Use:

- Reminder ads.
- Trust/FAQ angle.

## Add To Cart

Audience:

```text
META_RT_AddToCart_14D
```

Rules:

- AddToCart event.
- 14-day window.
- Exclude purchasers.

Use:

- Direct response.
- Shipping clarity.
- Reusable/no more lint roller waste angle.

## Video Viewers

Audience:

```text
META_RT_VideoViewers_7D
```

Rules:

- People who watched 25% or more of FurLift videos.
- 7-day window.

Use:

- Product demo.
- Before/after.
- Checkout CTA.

## Retargeting Budget

Start only after there is enough audience volume:

```text
NGN 1,000-2,000/day
```

Do not retarget tiny audiences aggressively.
