# Low-Budget Launch Plan

Budget: NGN 50,000

## Split

| Bucket | Budget | Trigger |
| --- | ---: | --- |
| TikTok Promote test | NGN 5,000 | Only if Promote works without tax blocker |
| Paid creator posts | NGN 20,000 | Start immediately |
| Meta/Instagram test | NGN 15,000 | After Meta pixel events verify |
| Reserve | NGN 10,000 | Only after traffic source shows sessions/ATC |

## First 48 Hours

### Day 1

1. Post 5 videos organically.
2. DM 20 creators.
3. Try TikTok Promote on the best post with NGN 2,000.
4. If Promote blocks, move that budget to creators.

### Day 2

1. Buy 2-4 creator posts.
2. Set up Meta fresh account and pixel.
3. Verify Meta PageView/AddToCart.
4. Launch Meta test only if tracking works.

## Meta First Test

Daily cap:

```text
NGN 3,000-5,000
```

Objective:

- Sales/conversions if Purchase event is verified.
- Traffic/landing-page views if only PageView/AddToCart are verified.

Creatives:

- `furlift-spark-a-problem-first.mp4`
- `furlift-spark-b-transformation-first.mp4`
- `furlift-spark-c-pov-style.mp4`

## Kill Rules

- CTR below 0.8% after enough impressions: stop creative.
- CPC above USD 1 equivalent with no add-to-cart: stop creative.
- 100 sessions with add-to-cart under 3%: fix landing/creative match.
- 3 checkout starts and no purchase: inspect checkout/payment trust.
- Any ad account compliance warning: pause and resolve before spending more.

## Scale Rules

- Do not scale on views alone.
- Scale only after sessions and add-to-cart.
- Increase budget by 20-30%, not 2x-5x jumps.
- Reuse winning creator comments/hooks in next creative batch.
