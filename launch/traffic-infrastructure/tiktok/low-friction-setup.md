# TikTok Low-Friction Setup

## Position

The current Shopify-created TikTok Ads Manager is blocked. TikTok's own advertiser review process can prevent ad publishing until account information is approved, and verification status may affect the ability to post ads.

The low-friction TikTok path is therefore:

1. Use the existing TikTok profile for organic posting.
2. Try TikTok Promote from the TikTok app.
3. Use paid creator posts.
4. Create a fresh individual advertiser account only if TikTok offers a truthful personal/individual flow.

## Path A: TikTok Promote

Use this from the TikTok mobile app, not Shopify.

Steps:

1. Post the top 5 FurLift videos organically.
2. Wait 4-12 hours.
3. Open the best-performing post.
4. Use Promote.
5. Choose website visits if available.
6. Use the UTM URL below.
7. Start with NGN 2,000.

URL:

```text
https://mxu168-9g.myshopify.com/products/furlift-reusable-pet-hair-detailer?utm_source=tiktok&utm_medium=promote&utm_campaign=furlift_promote_test&utm_content=best_video_01
```

Stop if TikTok asks for the same business/tax validation.

## Path B: Fresh Individual Advertiser Flow

Use only truthful operator details.

Allowed:

- Personal name.
- Real country.
- Real address.
- Individual Tax ID only if required and legitimately retrieved.
- Personal ad account if TikTok allows it.

Not allowed:

- Fake US identity.
- Fake business address.
- VPN country mismatch.
- Repeated account creation after rejection.

## Path C: Creator Posts

Use creators as paid distribution without the blocked Ads Manager.

Launch budget:

- NGN 20,000 for 4-8 micro-creator posts.
- NGN 2,500-5,000 per test post.

Creator requirements:

- Pet/home/cleaning content.
- Real English-speaking engagement.
- Will include `#ad`.
- Can place link in bio or caption.

Tracking URL pattern:

```text
https://mxu168-9g.myshopify.com/products/furlift-reusable-pet-hair-detailer?utm_source=tiktok_creator&utm_medium=paid_creator&utm_campaign=furlift_seed&utm_content={{creator_handle}}
```

## Go / No-Go

| Signal | Action |
| --- | --- |
| Promote accepts website traffic without tax blocker | Spend NGN 2,000 test |
| Promote asks for tax validation | Stop and use creator posts |
| Creator post drives Shopify sessions | Buy one more post from same creator |
| Creator post drives ATC | Reserve retargeting budget |
| No sessions from creator | Do not rebuy |
